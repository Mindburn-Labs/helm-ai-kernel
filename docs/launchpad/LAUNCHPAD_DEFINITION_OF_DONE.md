# Launchpad Definition Of Done

Launchpad is not complete until all of the following are true:

- [KEEP] `CODEX_CAPABILITY_AUDIT.md` and `LAUNCHPAD_REPO_AUDIT.md` exist.
- [KEEP] Launch CLI builds.
- [KEEP] Matrix and plan commands work.
- [REBUILD] OpenClaw local-container e2e passes.
- [REBUILD] Hermes local-container e2e passes or is blocked with verified reason.
- [KEEP] Codex is classified as external/BYO unless full redistribution and e2e proof exists.
- [REBUILD] Missing model key produces ESCALATE, not crash.
- [REBUILD] Unknown MCP tool quarantines.
- [REBUILD] Teardown emits receipt.
- [REBUILD] EvidencePack verifies offline.
- [KEEP] No app is marked Available without full conformance.
- [REBUILD] Enterprise route registry and OpenAPI parity pass.
- [REBUILD] Console builds and Playwright Launchpad e2e passes or is explicitly deferred.
- [KEEP] Docs do not claim more than tests prove.
- [REBUILD] Security review completed.
- [DEFER] Homebrew naming conflict plan exists.
